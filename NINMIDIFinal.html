Recipes and Help

Mouse Controls
  • Click + Drag (piano keys) Draw notes horizontally
  • Click (piano roll) Add/toggle note at position
  • Right-click (note) Delete note
  • Scroll wheel Zoom in/out on timeline
  • Shift + Click Select multiple notes
  • Click + Drag (knobs) Turn effect knobs
  • Double-click (knob) Reset to default value
  • Click (pad) Trigger sound for that drum
Keyboard Shortcuts
  • Space Play or pause
  • G Generate pattern
  • Shift+G Generate full song
  • R Random album
  • C Controlled chaos
  • S Save snapshot
  • Z Undo (history back)
  • X Redo (history forward)
  • D Toggle drum track lock
  • B Toggle bass track lock
  • M Toggle melody track lock
  • P Toggle pad track lock
  • A Toggle arpeggio track lock
  • 1-5 Select drum component (kick/snare/hihat/perc/tom)
  • Arrows Navigate grid
  • Delete Clear selected notes
  • Escape Stop playback
  • Ctrl+S Quick save to bank
  • Ctrl+E Export MIDI
Workflow Tips
  • Track Locks Lock a lane before generating to keep the pieces you like while mutating the rest. Perfect for building a song layer by layer.
  • Snapshots Use snapshots for named keeper ideas, and history for quick A/B auditions.
  • Sections Mark history items as SEC A and SEC B to stitch together a full arrangement for MIDI export.
  • Humanize Adjust velocity and timing variance to add organic feel to robotic patterns.
  • Controlled Chaos Use C to apply random mutations while respecting locks—great for finding variations.
  • Clone Lane Generate one track, then lock it and generate another to base new patterns on existing ones.
